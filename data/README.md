# Data Directory

This directory is for lightweight pointers and local symlinks only. Do not commit datasets.

Recommended layout:

- `videodpo/`: symlink or mount point for the original VideoDPO preference data.
- `youtubevos/`: symlink or mount point for YouTube-VOS-derived frames/masks.
- `generated_losers/`: offline loser videos and manifests generated by data ablations.

Prefer these environment variables in scripts:

- `VIDEO_DPO_DATA_ROOT`
- `YOUTUBE_VOS_ROOT`
- `GENERATED_LOSER_ROOT`

Existing local data may live elsewhere, such as `data/external/`; keep compatibility with older scripts by using path config instead of moving large files.

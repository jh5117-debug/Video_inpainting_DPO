# official_videodpo_diffueraser_data_partialmask_loser_comp_k4

Data-only ablation.

- win: original VideoDPO winner/chosen video.
- masks: offline K=4 random partial masks per win video.
- raw loser: generated by a video inpainting model from `(win, M_gen)`.
- comp loser: `win * (1 - M_gen) + raw_loser * M_gen`, with actual polarity checked per model.
- final loser: comp loser.
- training later: still official VideoDPO / DiffuEraser full-mask bridge.
- changed variable: data only.

Use `scripts/pai_generate_partialmask_losers_k4.sh` to create the planned
manifest schema and output folders. Full generation only happens after model
smoke tests pass.

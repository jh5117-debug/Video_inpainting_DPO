# Open This First

This folder is a lightweight advisor-facing visual evidence pack. It intentionally avoids raw caches and full generation roots.

## Suggested Meeting Order

1. DiffuEraser selected cases:
   `diffueraser/selected_8/exp11_outer_b075_s2_selected_visuals`

2. VideoPainter Step2000 shadow-dev examples:
   `videopainter/step2000_shadowdev/*step2000_side_by_side.mp4`

   `videopainter/step50_searchdev`

4. Use `index.csv` for exact file paths and why each asset is included.

## Do Not Browse During Meeting

Do not browse raw experiment roots or frame-cache folders during the meeting. This folder is the curated lightweight entry point.

## Claim Boundary

DiffuEraser and VideoPainter are the paper-positive evidence tracks. VideoPainter is a controlled low-data cross-pipeline validation, not a large-scale VideoPainter training dataset and not universal generalization.

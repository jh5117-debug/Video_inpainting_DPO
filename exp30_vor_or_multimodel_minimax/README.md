# Exp30 VOR-OR Multi-Model MiniMax

Exp30 is an isolated track for building a VOR-OR multi-model medium-hard
preference pool, testing MiniMax as a flow-style third adapter candidate, and
planning paper-ready three-backbone evidence.

The track starts from Exp29. It must not modify Exp1-Exp28 history, shared
trainers, or `inference/metrics.py`.
